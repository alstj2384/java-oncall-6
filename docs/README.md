### 근무일 배정 규칙

- [x]  순번에 따라 비상 근무일을 배정한다.
- [x]  평일, 휴일 각각 1회씩 편성되어야 한다.
- [x]  2일 연속 근무하게 될때면, 해당 순번(평일 or 휴일)의 다음 근로자와 순서를 바꾼다.
- [x]  법정 공휴일은 **휴일** 순번에 해당한다.

### 이름 요구사항

- [x] 이름은 콤마로 구분하여 좌우 공백을 무시한다.
- [x]  닉네임은 중복되면 안 된다.
- [x]  최대 5자로 유지해야 한다.
    - [x]  5자를 초과할 경우 예외가 발생한다.
- [x]  근무자는 5명 이상, 35명 이하로 관리되어야 한다.
    - [x]  범위를 벗어나는 경우 예외가 발생한다

### 입력

- [x]  월(숫자)와 요일 정보를 입력받는다.
- [x]  2월은 28일까지만 있다고 가정한다.
- [x]  비상 근무자 입력 과정 예외 발생 시 평일 근무부터 다시 받는다.

### 출력

- [x]  비상 근무표를 출력한다.
    - [x]  평일이면서 법정공휴일이면 요일 뒤에(휴일)을 표기한다.
- [x]  비상 근무표는 입력받은 월에 한해서 출력한다.
